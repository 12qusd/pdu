pduMain.py

Requires a PDUcomms() object to be instantiated and has the following methods available:

turnPortOn(self, port, ip) - Purpose: turn a given port at an IP on

  • Takes in port (int) and ip (string)
  • Outputs a tuple with the following data (port, ip, data)
    • port (int) is port number that was affected
    • ip (string) is the ip address that was affected
    • data (string) is some data associated with the success or failure of the function (probably just returns "on")

turnPortOff(self, port, ip) - Purpose: turn a given port at an IP off

  • Takes in port (int) and ip (string)
  • Outputs a tuple with the following data (port, ip, data)
    • port (int) is port number that was affected
    • ip (string) is the ip address that was affected
    • data (string) is some data associated with the success or failure of the function (probably just returns "off")

isOn(self, port, ip) - Purpose: tell whether a port at an ip is on now

  • Takes in port (int) and ip (string)
  • Outputs a tuple with the following data (port, ip, data)
    • port (int) is port number that was affected
    • ip (string) is the ip address that was affected
    • data (boolean) will tell if port is on or off (True - on, False - off)

readTemp(self, ip) - Purpose: read the temperature of a pdu

  • takes in an ip (string)
  • Outputs a tuple with the following data (ip, heat)
    • ip (string) is the ip address that was queried
    • heat (int) is the temperature of the heat unit associated with the ip

readCurrent(self, port, ip) - Purpose: read the current of a port

  • takes in a port (int) and ip (string)
  • Outputs a tuple with the following data (port, ip, data)
    • port (int) is the port queried
    • ip (string) is the ip address queried
    • data (float) is the current running through the port

readAllPorts(self, ip) - Purpose: read all ports on an ip at once

  • Takes in an ip (string)
  • Outputs an array of tuples each with the following data (port, ip data)
    • port (int) shows what port was queried for that array entry
    • ip (string) the ip that was queried
    • data (boolean) telling whether a port is on or off

interfacing.js

This code interfaces with the python code, but requires an RPC client to be passed into each of the functions. This code can be added to any node.js server by using the following code:

const interfacing = require('./path/to/code/interfacing.js')

Then the following methods can be invoked on the interfacing object

turnPortOn(portNum, ip, client, callback) - Purpose: turn a port on

  • Parameters:
    • portNum (int) number of the port
    • ip (string) ip address of pdu
    • client (object) rpc client that is connected to python spawn
    • callback (function) function to be called upon completion of this function
  • Function gets data returned from python through the response object (res)
